hxdz_try 
- Simple standalone script
- Try - Shows if your /do action is true or not - randomly generates answers: `YES` or `NO`
- Coin - Randomly selects between `Tails` and `Heads`

Translations `en` & `cs`

Translations are changeable in `client.lua`
ex.
```
HxDz.setLang('en')
```
or
```
HxDz.setLang('cs')
```
